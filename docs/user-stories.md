# User Stories

## High Priority (v1)

1. **Log Workout**
   > As a gym-goer, I want to log exercises with weight, sets, and reps so that I can track progress.

2. **View Progress**
   > As a gym-goer, I want to see charts of my lifts over time so that I know if I'm getting stronger.

3. **Reusable Routines**
   > As a gym-goer, I want to create and save routines (Push, Pull, Legs) so that I don't re-enter exercises every time.

4. **Edit/Delete Workouts**
   > As a gym-goer, I want to edit or delete past workouts so that I can fix mistakes.

## Nice-to-Have (v2)

5. **Share Routines**
   > As a gym-goer, I want to share routines with my gym buddy so that he can see what I'm doing.

6. **Show Trainer Progress**
   > As a gym-goer, I want to show my progress to my personal trainer.

---

## Out of Scope

- Social features
- Meal tracking
- Wearable integration

---

## MVP Features (v1)

- Log exercises (weight, sets, reps)
- Progress charts
- Reusable routines
- Edit/delete workouts

---

## Success Metrics

- Logs workout in <30 seconds
- Can see progress charts
