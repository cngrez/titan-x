### Current Workflow
1. Before gym: Decide routine (Push, Pull, Legs)
2. At gym: Open Notes app
3. Type: `Exercise – Weight × Reps, Reps, Reps`
4. Save note when done
5. Later: Scroll through old notes to manually compare progress

### Stakeholders
| User | Role |
|------|------|
| Client | Primary user (logs workouts) |
| Gym buddy | Occasionally asks to see routines/weights |
| Personal trainer | Occasionally asks to see progress |

### Tools Used
| Tool | Purpose |
|------|---------|
| Apple Notes | Logging workouts |
| Timer app | Rest periods |
| Spotify | Music |
| YouTube | Form checks |

### Client Goals
- Become stronger
- Stay motivated
- See progress over time

### Top Pain Points
1. **Manual progress tracking** - Scrolling through old notes to compare weights
2. **No structure** - Notes app is messy and disorganized
3. **No automatic insights** - Wants app to tell them "Last time you lifted X"

---

## Out of Scope (Client not necessary)
-  Social features (friends, competition)
-  Meal tracking / calorie counting
-  Wearable integration

---

## Success Metrics
- Can view progress charts per exercise
- Can log a workout in under 30 seconds
